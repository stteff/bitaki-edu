# bitaki-edu
Εκπαιδευτικό υλικό για την πλακέτα ανοιχτής τεχνολογίας, BBC micro:bit
Περιλαμβάνει Δραστηριότητες στο περιβάλλον `makecode` της Microsoft σε μορφή φύλλων εργασίας, καθώς και τα αντίστοιχα προγράμματα
